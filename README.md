# NutriAbuelita

Aplicación web estática para hacer seguimiento familiar al plan de alimentación semanal de una abuelita.

## Cómo usar

1. Descomprime el ZIP.
2. Abre `index.html` en Chrome, Edge o cualquier navegador moderno.
3. La información se guarda en el navegador usando `localStorage`.
4. Puedes subir la carpeta a GitHub Pages, Netlify o Firebase Hosting si quieres verla desde varios dispositivos.

## Incluye

- Plan del día.
- Seguimiento diario de comidas, agua, apetito, energía y alertas.
- Menú semanal sugerido.
- Armador de almuerzo.
- Postrecitos permitidos para rotar.
- Lista de mercado.
- Exportar/importar backup del seguimiento.
- Imprimir plan.

## Importante

La app organiza el plan familiar y ayuda a recordar reglas de alimentación baja en sodio y grasa. No modifica indicaciones médicas específicas.
